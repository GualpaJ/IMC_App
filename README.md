# 🏋️‍♂️ IMC Calculator App – Android Kotlin

**Descripción:**  
Aplicación nativa de Android desarrollada en Kotlin que permite calcular el **Índice de Masa Corporal (IMC)** a partir de la altura y el peso del usuario. La app muestra el resultado numérico junto con su clasificación y cambia dinámicamente el color del CardView según el rango. Es un proyecto práctico para aprender desarrollo Android y manejo de UI interactiva con Material Design.

---

## 📌 Funcionalidades

- Ajustar la altura mediante un **Slider**.  
- Incrementar o disminuir el peso con botones.  
- Calcular el IMC con un botón dedicado.  
- Mostrar resultado en un **CardView** con color y texto según el rango de IMC:  
  - Azul → Bajo peso  
  - Verde → Peso normal  
  - Amarillo → Sobrepeso  
  - Rojo → Obesidad  
- Interfaz simple, clara y responsiva.

---

## 🛠 Tecnologías utilizadas

- **Kotlin**  
- **Android Studio**  
- **Material Design Components** (MaterialCardView, Slider, Buttons)  
- Layouts: `LinearLayout`, `CardView`, `TextView`, `Button`
